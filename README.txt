Программа скриншотит ленту любого открытого профиля по ссылке на сайте vk.com и сохраняет файлы в отдельную папку с именем - идентификатором. 
Требует аутентификации в социальной сети для работы. Код открыт - можете убедиться, что персональные данные никуда не утекают. 

Для запуска программа требует интерпретатор кода на Python. 
Требует так же установки библотеки для автоматизации работы с браузером selenium и библиотеки для работы с GUI через код Python - pyautogui.

Создана в рамках практики на курсе "WEB Парсинг на Python". Отдельное спасибо автору курса Павлу Хошеву.
